# Smart Grid bezpečnost

## Koncept Smart Grid

[[imork-smart-grid|Smart Grid]] představuje chytrou elektrickou síť s klíčovými vlastnostmi:

- **Obousměrný tok energie** — spotřebitelé se stávají i výrobci (prosumers)
- **Decentralizované řízení** — distribuovaná správa místo centralizované
- **Hierarchická struktura** — více úrovní řízení a komunikace

## Komunikační technologie

Smart Grid využívá různé komunikační technologie pro propojení jednotlivých komponent sítě (PLC, bezdrátové sítě, optická vlákna apod.).

## Normy a standardy

### IEC 62351

Standard pro bezpečnost v prostředí smart grid — definuje bezpečnostní mechanismy pro komunikační protokoly energetických systémů.

### NISTIR 7628

Třísvazkový dokument pro kybernetickou bezpečnost smart grid:

1. **Svazek 1** — Strategie kybernetické bezpečnosti
2. **Svazek 2** — Ochrana soukromí (privacy)
3. **Svazek 3** — Podpůrné analýzy

## 7 domén Smart Grid

NIST definuje 7 vzájemně propojených domén smart grid (výroba, přenos, distribuce, spotřeba, trh, operátor, poskytovatel služeb).

## Bezpečnostní přístupy

### Defense-in-depth

Vrstvená obrana — více bezpečnostních vrstev zajišťuje, že selhání jedné neohrozí celý systém.

### GWAC Architecture Model

GridWise Architecture Council — referenční model pro interoperabilitu smart grid systémů.

## EU vize

Evropská unie prosazuje smart grid jako klíčový prvek energetické transformace a dekarbonizace.

## ENISA — Communication Network Interdependencies (2015)

Analýza vzájemných závislostí komunikačních sítí v kontextu smart grid:

- **Kaskádové efekty** — selhání jedné komponenty může vyvolat řetězovou reakci
- **Dobré praktiky** — doporučení pro minimalizaci rizik vzájemných závislostí
